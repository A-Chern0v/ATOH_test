# Тестовое задание на аналитика для АТОН

Проведена очистка и нормализация исходного датафрейма в соответствии с поставленными условиями.
Также был проведён анализ данных: найдены компании, в которых суммарная доля владельцев превышала 100%, компании, где доли менялись со временем, а также найдены владельцы, участвующие более чем в одной компании.

По результатам проведенного исследования был составлен небольшой текстовый отчёт.
